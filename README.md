# Tesla-Accident-Data-Analysis
This project focuses on analyzing Tesla-related accident data to uncover patterns in fatal incidents, vehicle interactions, and the role of Autopilot. Through data cleaning, exploratory data analysis (EDA), and statistical insights, the project aims to provide a deeper understanding of safety trends associated with Tesla vehicles.

---

### 🧹 1. Data Cleaning & Preprocessing

* Performed initial data inspection:

  * Checked data types and consistency
  * Identified and handled missing values
  * Removed duplicate records
* Dropped irrelevant or redundant columns to streamline analysis

---

### 📊 2. Exploratory Data Analysis (EDA)

#### 📅 Temporal Analysis

* Examined number of events:

  * By date
  * Year-wise trends
  * Daily distribution
* Compared event frequency across different states and countries

---

#### ⚠️ Accident & Fatality Insights

* Analyzed key aspects of fatal incidents:

  * Number of victims per accident
  * Total number of Tesla driver fatalities
  * Proportion of accidents involving one or more deaths

---

#### 🚶‍♂️ Collision Analysis

* Investigated accident types:

  * Incidents involving pedestrians and cyclists
  * Cases where both Tesla occupants and pedestrians/cyclists were killed
  * Frequency of Tesla collisions with other vehicles

---

#### 🚘 Model-wise Analysis

* Studied accident distribution across Tesla models
* Identified trends and variations between different vehicle models

---

#### 🤖 Autopilot Analysis

* Examined distribution of verified Tesla Autopilot-related deaths
* Evaluated patterns and frequency of such incidents

---

### 🛠️ Tools & Technologies

* Python (Pandas, NumPy)
* Data Visualization (Matplotlib, Seaborn)
* Jupyter Notebook

---

### 📈 Key Outcomes

* Identified temporal and geographical accident trends
* Quantified risks associated with different accident types
* Provided insights into Autopilot-related fatalities
* Highlighted model-specific patterns in accident data
